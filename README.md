# IMDB Sentiment Analysis

## Objective
Classify movie reviews as positive or negative using classical NLP and transformer-based NLP.

## Dataset
IMDB Dataset of 50K Movie Reviews

## Models
- TF-IDF + Logistic Regression
- DistilBERT

## Results
- TF-IDF + Logistic Regression: 89%
- DistilBERT: 90.45%

## Tech Stack
Python, Pandas, Scikit-learn, PyTorch, Hugging Face Transformers, Google Colab

## Key Learnings
- Text cleaning
- TF-IDF
- Logistic Regression
- Tokenization
- Fine-tuning DistilBERT
